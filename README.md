# IndigoSandbox

**From night sketches to living AI-products**

Сайт с кейсами по реализации AI-проектов. У каждого кейса: как было → как стало → эффект (экономия) → стек → применение в других отделах → кто обучился.

- Тёмная тема в стиле Notion
- Адаптивная вёрстка

## Просмотр

Откройте `index.html` в браузере или включите [GitHub Pages](https://docs.github.com/en/pages) для этого репозитория (Settings → Pages → Source: main branch).

## Локально

Клонирование и просмотр:

```bash
git clone https://github.com/YOUR_USERNAME/IndigoSandbox.git
cd IndigoSandbox
# открыть index.html в браузере
```
